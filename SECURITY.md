# Security Policy

Security bugs in Factory Island are taken seriously.
We appreciate responsible disclosure and will acknowledge contributions.

## Scope

Bugs in this repository are in scope.
Bugs in third-party dependencies (React, Phaser, TypeScript, etc.) are not in scope
unless they result in a Factory Island specific vulnerability.

## Supported Versions

Only the latest release gets security updates.

## Reporting a Vulnerability

Please report vulnerabilities via GitHub Issues (private security advisory)
or contact the maintainer directly.

Include:
- Steps to reproduce
- Browser version
- Operating system
- Expected vs. actual behavior
